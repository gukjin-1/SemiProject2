## 목표

하루에 먹는 음식을 찍어 영양소 분석 후 일일 권장량에 따라 음식을 추천해주는 서비스를 이용해 건강하게 관리하는 모델 만들기



## 프로젝트 수행 과정

1. 음식 사진 및 영양소 데이터 수집
2. yolo 모델 구축 
3. yolo 모델에서 검출 및 분류
4. 분류된 값을 일일 권장량과 비교해 부족하거나 넘칠 때 같은 카테고리 안에 있는 음식 추천
